# getpod-cli

GetPod CLI is a command-line tool for managing pods and containers.

## Installation

```bash
go install github.com/CloudCraft-Studio/getpod-cli@latest
```

## Development

### Prerequisites

- Go 1.21+
- golangci-lint (for linting)

### Build

```bash
make build
```

### Run

```bash
make run
```

### Test

```bash
make test
```

### Lint

```bash
make lint
```
